## 本地快速体验
通过该应用，您可以简单快速的创建一个 FG 案例到华为云函数计算服务。

- 下载命令行工具：`npm install -g @serverless-devs/s`
- 初始化一个模版项目：`s init start-fg-http-nodejs18`
- 进入项目后部署项目：`cd start-fg-http-nodejs18 && s deploy`
